# ReactLearning01# React Learning 01

ยินดีต้อนรับสู่โปรเจกต์แรกสำหรับการเริ่มต้นเรียนรู้ **React.js**!

## วัตถุประสงค์ (Goals)
โปรเจกต์นี้จัดทำขึ้นเพื่อใช้ในการฝึกฝนพื้นฐานของ React โดยเน้นไปที่:
*   การเข้าใจโครงสร้างของโปรเจกต์ (Folder Structure)
*   การสร้างและใช้งาน Components
*   การจัดการ State และ Props
*   การทำความเข้าใจ JSX

## เครื่องมือที่ใช้ (Tech Stack)
*   **React** (v18+)
*   **Vite** หรือ **Create React App**
*   **Git / GitHub** สำหรับการคุมเวอร์ชัน
